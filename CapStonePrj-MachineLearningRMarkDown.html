Finding correlations between different variables.

Key observations are cylinders having moderate -ve correlation with fuel consumption , i.e higher the number of cylinders, lower the fuel efficiency. So, cars with lower number of cylinders give more mile per gallon.

Cylinders and engine displacement show positive correlation. So, Engine displacement correlation to city fuel is similar to Cylinders. Higher the engine displacement, lower is the fuel efficiency.

CityFuel and gears show moderate negative correlation. Higher the number of gears, lower the fuel efficiency.

city fuel and annual fuel cost show moderate to good -ve correlation. The fuel cost spent on cars with higher mile per gallon is lower than fuel cost spent on cars with lower miles per gallon.

City fuel and GHG rating show moderate positive correlation. Higher the mile per gallon, higher the GHG rating (lower the emissions). Higher GHG rating is better for the environment.

From the observations and correlations, machine learing problem on this dataset is a supervised problem. The fuel usage will be predicted based on the features identified. The features are all part of the manfacturer configuration - cylinders, engine displacment and transmission. Also added gears that showed moderate correlation with city fuel. The machine learing techique used is linear regression. The success will be evaluated based on the linear regression results. If the results show the correlation is not by chance and show significant difference. Also, all the linear regression assumptions of linearity, normality , homoscedasticity should be met.

Linear model of all variables on mpg

Summary of Machine learning Analysis

Linear regression model was chosen to show the impact of selected features on City Fuel. The manufacture configuration includes Cylinders, Engine displacement, transmission type. So selected these as the features for the dependent variable City Fuel. Cylinders was excluded in the model, as it shows multi collinearity with Engine displacement. Engine Displacement shows high negative correlation (-0.83) with City Fuel. Gears is another feature in multiple regression as it shows moderate correlation with city fuel.

  • All 3 features run together in the model showed siginificant and better impact (72%) on city fuel, compared to each of them individually. The model would have been a good fit if all the assumtions of linear model are met. But the model shows some non linearity and skewness in normality. Although the model shows significance, we cannot use the model to predict fuel usage as it did not meet all assumptions of linear model. Non parametric regression can be used on existing data set as it does not look for linearity.
